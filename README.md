# 🍽️ Rohini's Restaurant

A modern and responsive restaurant website built using HTML and CSS.  
This project showcases a premium restaurant UI with elegant design, food sections, gallery, and booking form.

---

## 🚀 Features

✨ Responsive Navbar  
✨ Hero Section  
✨ Veg Special Food Cards  
✨ About Section  
✨ Food Gallery  
✨ Booking Form  
✨ Hover Effects & Animations  
✨ Modern Dark Theme UI  

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
